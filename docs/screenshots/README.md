# Screenshots

App screenshots referenced from the top-level `README.md` / `README.ja.md`.

- `notes.png` — notes list / cards
- `ai-actions.png` — note editor with on-device AI actions
- `settings.png` — Settings → AI model, theme, language, Wi-Fi-only download

Captured on an Android emulator (`adb exec-out screencap -p > out.png`). To
refresh, recapture at a similar resolution and overwrite these files. On iOS use
`xcrun simctl io booted screenshot out.png`.
